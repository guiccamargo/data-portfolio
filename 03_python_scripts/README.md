# 🐍 Python Scripts — Portfolio

Coleção de projetos Python cobrindo desenvolvimento web, automação, integração com APIs, análise de dados e criação de jogos. Cada projeto foi desenvolvido com foco em aplicações práticas e reais.

---

## 🗂️ Projetos por categoria

### 🌐 Desenvolvimento Web (Flask)

| Projeto | Descrição | Tecnologias |
|---------|-----------|-------------|
| [cafe_and_wifi](./cafe_and_wifi) | Website para cadastrar e consultar os melhores cafés de uma cidade | Flask, SQLite |
| [lord_of_the_rings](./lord_of_the_rings) | Consulta de informações sobre a saga via The One API | Flask, Bootstrap, REST API |
| [online_shop](./online_shop) | Loja virtual com carrinho e pagamentos | Flask, SQLAlchemy, Stripe API |
| [to_do_list](./to_do_list) | Aplicativo de lista de tarefas | Flask, HTML, CSS, Bootstrap, SQLite |
| [portfolio_website](./portfolio_website) | Template de website para portfólio pessoal | Flask, HTML, CSS |
